# Calculadora-de-Pre-o-Total
Programa que calcula o preço total de uma compra

## Sobre o programa
Este programa em Python calcula o valor total de uma compra a partir de informações simples do produto.  
Ele mostra o nome do item, o preço unitário, a quantidade comprada e o valor final, tudo de forma clara e fácil de entender.

## Como funciona
- O programa define o nome do produto, o preço unitário e a quantidade.
- Em seguida, calcula o preço total multiplicando o valor unitário pela quantidade.
- Por fim, exibe todas as informações no terminal.
